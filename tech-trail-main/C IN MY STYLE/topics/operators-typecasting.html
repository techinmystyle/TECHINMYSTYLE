Type Casting in C

Introduction

Type casting in C is the process of converting one data type into another. It is crucial when performing operations between different data types to ensure accuracy and prevent unexpected behavior.

C supports two types of type casting:

  • Implicit Type Casting (automatic)
  • Explicit Type Casting (manual)

1. Implicit Type Casting

Also known as automatic type conversion, this occurs when the compiler automatically converts a smaller data type to a larger one (e.g., int to float) to prevent data loss.

implicit_cast.c
#include <stdio.h>

int main() {
    int i = 10;
    float f;

    f = i; // implicit conversion from int to float
    printf("Implicit cast: %f\\n", f);

    return 0;
}

2. Explicit Type Casting

Also known as manual casting, this requires the use of a cast operator. It is used when you want to convert data types forcefully and consciously.

Caution!

Improper explicit casting can lead to data loss or undefined behavior.

explicit_cast.c
#include <stdio.h>

int main() {
    float f = 3.14;
    int i;

    i = (int)f; // explicit conversion from float to int
    printf("Explicit cast: %d\\n", i);

    return 0;
}

Example Use Case

Consider this example where casting is needed to get a correct result:

division_cast.c
#include <stdio.h>

int main() {
    int a = 5, b = 2;
    float result;

    result = (float)a / b; // cast a to float
    printf("Result = %f\\n", result); // Output: 2.500000

    return 0;
}

Interview Questions

What is the difference between implicit and explicit type casting?

Implicit casting is done automatically by the compiler; explicit casting is manually done by the programmer using a cast operator like (int) or (float).

Why should we use explicit type casting in C?

Explicit type casting ensures controlled and predictable conversion, especially useful during operations like division or when assigning float to int.
