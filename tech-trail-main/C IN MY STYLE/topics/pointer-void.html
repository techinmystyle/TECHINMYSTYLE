Void Pointers in C

Overview

A void pointer, also known as a generic pointer, is a special type of pointer that can point to any data type. It is declared using the void * syntax and is useful when the type of data is unknown or flexible during compilation.

Key Point

Void pointers cannot be dereferenced directly without type casting.

Syntax and Example

Here's a basic usage of a void pointer:

void_pointer.c
#include <stdio.h>

int main() {
    int a = 10;
    float b = 3.14;
    void *ptr;

    ptr = &a;
    printf("Integer value: %d\\n", *(int *)ptr);  // Type cast required

    ptr = &b;
    printf("Float value: %.2f\\n", *(float *)ptr);  // Type cast required

    return 0;
}

Explanation:

  • void *ptr; declares a generic pointer.
  • ptr = &a; stores the address of an integer.
  • *(int *)ptr casts the void pointer to the correct type before dereferencing.

Real-World Applications

Generic Functions

Used in standard library functions like qsort() and bsearch().

qsort bsearch

Data Abstraction

Used in abstract data structures (like stacks, queues) to store any type of element.

Generic Stack Void Queues

Interview Questions

What is a void pointer?

A void pointer is a generic pointer that can store the address of any data type but must be type cast before dereferencing.

Can we perform pointer arithmetic on void pointers?

No. Since void pointers do not have a defined data type size, pointer arithmetic is not allowed without typecasting.

Where are void pointers commonly used?

They're commonly used in library functions, data structure implementations, and generic programming scenarios.
