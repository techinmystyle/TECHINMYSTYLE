Control Flow in C

Overview

Control flow in C refers to the order in which individual statements, instructions, or function calls are executed or evaluated. C provides constructs for decision making, looping, and branching to control the flow of the program.

Key Idea

Control structures allow programs to make decisions, repeat actions, and jump to specific parts of code based on logic.

Types of Control Flow

  • Decision Making: if, if-else, nested if, switch
  • Loops: for, while, do-while
  • Jump Statements: break, continue, goto, return

Example Programs

1. Decision Making (if-else)

#include <stdio.h>

int main() {
    int age = 18;
    if (age >= 18) {
        printf("You are eligible to vote.\\n");
    } else {
        printf("You are not eligible.\\n");
    }
    return 0;
}

2. Looping (for loop)

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("Number: %d\\n", i);
    }
    return 0;
}

3. Jump Statement (break)

#include <stdio.h>

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) break;
        printf("%d\\n", i);
    }
    return 0;
}

Real-World Applications

AI/ML Conditions

Decision trees and logic gates use if-else and switch cases.

Game Development

Loops and decisions control player health, scoring, and game progression.

IoT Devices

Microcontrollers use conditional loops to manage sensors and outputs.

Interview Questions

What is the difference between while and do-while?

while checks the condition before execution, do-while runs at least once before checking the condition.

Can we use multiple break statements in a loop?

Yes, multiple break statements can be used based on different conditions to exit a loop early.
