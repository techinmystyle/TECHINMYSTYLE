Introduction to File Handling in C

Overview

File handling in C allows us to store data permanently on a storage device. Unlike variables, which are temporary, files preserve data even after the program terminates. The C standard library provides a set of functions to create, open, read, write, and close files efficiently using the stdio.h header.

Key Point

Files in C are handled using FILE pointers and related functions like fopen(), fclose(), fprintf(), fscanf(), fread(), and fwrite().

Types of Files in C

  • Text Files: Store data in human-readable format. e.g., sample.txt
  • Binary Files: Store data in binary format. e.g., data.bin

Basic File Operations

fopen()

Opens a file in a specific mode (read, write, append, etc.).

fprintf(), fscanf()

Write and read formatted data to/from a file (text files).

fread(), fwrite()

Used for reading/writing binary data.

fclose()

Closes an opened file and frees resources.

Example: Creating and Writing to a File

#include <stdio.h>

int main() {
    FILE *fp = fopen("example.txt", "w");

    if (fp == NULL) {
        printf("Error opening file!\\n");
        return 1;
    }

    fprintf(fp, "Hello, file handling in C!\\n");
    fclose(fp);

    return 0;
}

Code Breakdown:

  • fopen() opens the file example.txt in write mode.
  • fprintf() writes text to the file.
  • fclose() safely closes the file.

Real-World Applications

Log Files

Store application logs, user activities, and error tracking.

Data Storage

Persist user preferences, game scores, or analytics in files.

Configuration Files

Read and write configuration settings for system/software.

Interview Questions

What is the use of FILE* in C?

FILE* is a pointer to a file structure used by the C standard library to keep track of file streams.

What happens if you forget to close a file using fclose()?

It may lead to data loss, resource leaks, or file corruption depending on the system and operation performed.

What is the difference between text and binary files?

Text files store data in readable characters, while binary files store in raw byte format, making them more compact and faster to read/write.
