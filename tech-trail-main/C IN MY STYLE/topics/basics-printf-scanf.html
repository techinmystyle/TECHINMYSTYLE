Input and Output in C

Overview

Input and Output (I/O) operations are fundamental in C programming for interacting with users and systems. C uses functions from the <stdio.h> library such as printf() and scanf() for standard output and input respectively.

Fun Fact

The printf and scanf functions have been part of C since its inception in the 1970s!

Standard Output: printf()

printf() is used to display output on the screen.

output.c
#include <stdio.h>

int main() {
    int age = 20;
    printf("I am %d years old.\\n", age);
    return 0;
}
  • %d – Format specifier for integers
  • %f – For floating-point numbers
  • %c – For single characters
  • %s – For strings

Standard Input: scanf()

scanf() is used to get input from the user.

input.c
#include <stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);
    printf("You entered: %d\\n", age);
    return 0;
}
  • &age – The address-of operator is used to pass variable address to scanf()
  • Always match the format specifier with the variable type

Multiple Inputs/Outputs

multi_io.c
#include <stdio.h>

int main() {
    int a, b;
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);
    printf("Sum = %d\\n", a + b);
    return 0;
}

Real-World Applications

CLI Tools

Console-based tools use I/O to take commands and display results.

User Input Handling

Survey or login systems gather data using scanf().

System Monitoring

Programs output logs and read commands via I/O.

Interview Questions

What is the purpose of & in scanf()?

& is used to pass the memory address of a variable so that scanf() can store the input value in that variable.

Can you take multiple inputs in one scanf()?

Yes, by using multiple format specifiers and variables, like scanf("%d %d", &a, &b)
