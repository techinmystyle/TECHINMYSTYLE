Types of Functions in C

Overview

In C programming, functions are blocks of code that perform a specific task. Based on how they are defined and used, functions are classified into different types. These help in organizing code, reusability, and modular programming.

Key Insight

Functions improve program structure and reduce redundancy. They can accept inputs (parameters) and return outputs (return values).

Classification of Functions

  • Library Functions: Predefined in C standard libraries (e.g., printf(), scanf(), strlen()).
  • User-Defined Functions: Created by the programmer to perform custom tasks.

Types of User-Defined Functions

User-defined functions in C are categorized based on the presence or absence of arguments and return values:

1. No Arguments, No Return Value

Simple function that performs a task but neither accepts input nor returns output.

2. Arguments, No Return Value

Accepts inputs to process, but doesnâ€™t return a result.

3. No Arguments, Return Value

Doesn't take input but returns a result after internal processing.

4. Arguments and Return Value

Takes input parameters and returns a result. Most commonly used function type.

Example: Function with Arguments and Return Value

#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    printf("Sum: %d\\n", result);
    return 0;
}

Real-World Applications

Modular Code

Functions help split large programs into manageable parts.

Reusable Logic

Reusable functions in embedded systems, calculators, etc.

Interview Questions

What are the types of user-defined functions in C?

There are four types: no arguments no return, arguments no return, no arguments return value, and arguments with return value.

What is the difference between library and user-defined functions?

Library functions are pre-built (e.g., printf), while user-defined functions are written by the programmer.
