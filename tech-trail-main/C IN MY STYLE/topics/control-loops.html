Loops in C Programming

Overview

Loops in C allow executing a block of code repeatedly under certain conditions. They are essential for tasks involving repetition, such as iterating through arrays, performing computations, and automating repetitive logic.

Did You Know?

The first loop executes code without condition check is the do-while loop!

Types of Loops in C

  • for loop: Used when the number of iterations is known.
  • while loop: Used when the number of iterations is not known and depends on a condition.
  • do-while loop: Like while loop, but executes the body at least once.

Examples

1. for loop

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("Count: %d\\n", i);
    }
    return 0;
}

2. while loop

#include <stdio.h>

int main() {
    int i = 1;
    while (i <= 5) {
        printf("While Loop: %d\\n", i);
        i++;
    }
    return 0;
}

3. do-while loop

#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("Do-While Loop: %d\\n", i);
        i++;
    } while (i <= 5);
    return 0;
}

Real-World Applications

Games

Used to continuously check for player inputs and update game state.

Automation

Perform repetitive hardware instructions like blinking LEDs or reading sensor values.

Data Processing

Loops help in iterating over large datasets or input/output buffers.

Interview Questions

What is the difference between while and do-while loops?

while checks the condition before executing the loop, while do-while executes the loop once before checking the condition.

Can you use break and continue in loops?

Yes. break exits the loop prematurely. continue skips the current iteration and proceeds to the next.

How can we create an infinite loop?

Using: while(1), for(;;), or do {...} while(1);.
