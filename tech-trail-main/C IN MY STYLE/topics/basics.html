Basics of C Programming

Overview

Before diving into advanced topics, it's essential to understand the foundational elements of the C programming language. This includes understanding the structure of a program, tokens, keywords, identifiers, data types, variables, constants, and basic input/output.

Tip

Mastering the basics helps you build a strong foundation and understand how memory, logic, and compilation work in C.

Basic Building Blocks

  • Tokens: The smallest individual units in a C program (keywords, identifiers, constants, strings, operators, etc.).
  • Keywords: Reserved words like int, return, while.
  • Identifiers: Names used to identify variables, functions, arrays (e.g., count, main).
  • Constants: Fixed values that do not change during execution (e.g., 5, 'A', 3.14).
  • Variables: Named memory locations to store data.
  • Data Types: Define the type of data a variable can store (e.g., int, float, char).

Example Program

This program demonstrates basic syntax, declaration, assignment, and printing of variables.

basics.c 
#include <stdio.h>

int main() {
    int age = 20;
    float height = 5.9;
    char grade = 'A';

    printf("Age: %d\\n", age);
    printf("Height: %.1f\\n", height);
    printf("Grade: %c\\n", grade);

    return 0;
}

Interview Questions

What is a token in C?

A token is the smallest unit in a C program, such as keywords, identifiers, operators, constants, etc.

What are keywords in C?

Keywords are reserved words predefined by the C language (e.g., int, return, while).

Difference between constant and variable?

A variable can change its value during execution, whereas a constant's value remains fixed.
