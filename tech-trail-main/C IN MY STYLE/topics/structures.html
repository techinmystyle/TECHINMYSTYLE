Introduction to Structures and Unions in C

Overview

Both structures and unions in C are user-defined data types that allow grouping multiple variables of different data types under one name. They help in modeling complex data structures such as records, data packets, or components.

Although structures and unions are similar in syntax, they differ in memory allocation and usage. Structures allocate separate memory for each member, while unions share the same memory for all members.

Key Difference

In a structure, each member has its own storage, but in a union, all members share the same memory location.

Declaration Syntax

Structures and unions are declared in a similar way, but their behavior in memory is different.

structure_union.c
#include <stdio.h>

// Structure definition
struct Student {
    int id;
    char name[30];
    float marks;
};

// Union definition
union Data {
    int i;
    float f;
    char str[20];
};

Memory Difference

Let's understand how memory is used differently in structures and unions.

memory_demo.c
#include <stdio.h>

struct DemoStruct {
    int a;
    float b;
};

union DemoUnion {
    int a;
    float b;
};

int main() {
    struct DemoStruct s;
    union DemoUnion u;

    printf("Size of struct: %lu\\n", sizeof(s));  // Typically 8 bytes
    printf("Size of union: %lu\\n", sizeof(u));   // Typically 4 bytes

    return 0;
}

Explanation:

  • Struct: Total size = sum of all members (plus padding if any).
  • Union: Total size = size of the largest member only.

Real-World Applications

Student/Employee Records

Unions in Embedded Systems

Unions help to access hardware registers with multiple interpretations.

Microcontrollers Device drivers

Interview Questions

What is the difference between structure and union in C?

Structure allocates memory for each member separately, while union shares the same memory among all members.

When should we use a union instead of a structure?

Use unions when variables are mutually exclusive and memory efficiency is critical, such as in embedded or network programming.

Can a structure contain a union as a member?

Yes, a structure can contain a union and vice versa. This allows flexible and memory-efficient representations of data.
