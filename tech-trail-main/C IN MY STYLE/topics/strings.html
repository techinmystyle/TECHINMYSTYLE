Introduction to Strings in C

Overview

In C, a string is essentially a sequence of characters terminated by a null character '\0'. Unlike other programming languages, C does not have a built-in string type. Instead, strings are represented as character arrays.

Note

Always ensure that strings in C are null-terminated to avoid undefined behavior during string operations.

Declaration and Initialization

You can declare and initialize strings in two ways:

string_init.c
#include <stdio.h>

int main() {
    char str1[] = "Hello";
    char str2[6] = {'W', 'o', 'r', 'l', 'd', '\0'};

    printf("%s %s\\n", str1, str2);
    return 0;
}

Explanation:

  • char str1[] = "Hello"; - Automatically adds '\0' at the end.
  • char str2[6] = {...} - Manually includes the null character.
  • %s format specifier is used for printing strings.

String Input and Output

C provides functions like scanf(), gets(), puts(), and printf() to handle strings.

string_io.c
#include <stdio.h>

int main() {
    char name[50];

    printf("Enter your name: ");
    scanf("%s", name);  // Reads string until space

    printf("Hello, %s!\\n", name);
    return 0;
}

Interview Questions

What is a string in C?

A string is a character array terminated by a null character '\0'.

How are strings stored in memory?

Each character occupies one byte, with the last byte reserved for the null character.

How to read strings with spaces in C?

Use fgets() instead of scanf() to read strings with spaces.
