Two-Dimensional Arrays in C

Overview

A two-dimensional array in C is essentially an array of arrays. It is used to represent tabular data or matrices, where data is arranged in rows and columns.

2D arrays are ideal for scenarios such as mathematical computations, grid-based games, and storage of data in table format.

Key Concept

In memory, 2D arrays are stored in row-major order, meaning elements of each row are stored in contiguous memory locations.

Syntax

Declaration and initialization of a 2D array:

array2d_syntax.c 
// Declaration
int matrix[3][4]; // 3 rows, 4 columns

// Initialization
int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

// Accessing elements
printf("%d", matrix[1][2]);  // Output: 6

Example Program

This program inputs values into a 2x3 matrix and displays them in matrix form.

matrix_input_output.c 
#include <stdio.h>

int main() {
    int matrix[2][3], i, j;

    printf("Enter values for a 2x3 matrix:\\n");
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            scanf("%d", &matrix[i][j]);
        }
    }

    printf("Matrix is:\\n");
    for(i = 0; i < 2; i++) {
        for(j = 0; j < 3; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\\n");
    }

    return 0;
}

Real-World Applications

Data Tables

Storing rows and columns of data like spreadsheets

Game Boards

Used in grid-based games such as chess, Sudoku, and tic-tac-toe

Matrix Operations

Linear algebra applications like multiplication, addition, and transformation
