Switch Statement in C

Overview

The switch statement in C is a control statement that allows multi-way branching. It evaluates an expression and transfers control to the matching case label. It is an alternative to writing many if-else-if statements.

Use Case

Use switch when you want to compare a single variable to multiple constant values efficiently.

Syntax

switch (expression) {
    case constant1:
        // code block
        break;
    case constant2:
        // code block
        break;
    ...
    default:
        // default code block
}

Example

#include <stdio.h>

int main() {
    int choice = 2;

    switch (choice) {
        case 1:
            printf("Option 1 selected\\n");
            break;
        case 2:
            printf("Option 2 selected\\n");
            break;
        case 3:
            printf("Option 3 selected\\n");
            break;
        default:
            printf("Invalid choice\\n");
    }

    return 0;
}

Key Points

  • switch only works with integral types like int, char, enum.
  • Each case must have a constant value.
  • break is used to prevent fall-through to the next case.
  • default executes if no case matches.

Real-World Applications

Menu-driven programs

Efficiently handle user options in console interfaces.

Embedded Systems

Used in state machines or device command handling.

Command Dispatch

Based on a command code, perform specific operations.

Interview Questions

What happens if we omit break in a switch?

It results in a fall-through where control continues into the next case block even if it's not matched.

Can we use float values in a switch case?

No, only integral types like int, char, and enum are allowed.

Is default case mandatory?

No, but it's good practice to include it to handle unexpected values.
