Array Operations in C

Overview

Arrays in C support a range of common operations such as traversal, insertion, deletion, searching, and updating. Understanding these operations is essential for manipulating data efficiently in real-world applications.

Key Insight

Arrays in C have a fixed size, meaning operations like insertion and deletion often require manual shifting of elements.

Basic Operations

  • Traversal: Accessing each element using a loop
  • Insertion: Adding a new element at a specific position
  • Deletion: Removing an element from a specific position
  • Search: Finding the index of a given element
  • Update: Changing the value of an element at a specific index

Example Program

This program demonstrates traversal, insertion, deletion, and searching in a 1D array.

array_operations.c 
#include <stdio.h>

int main() {
    int arr[100] = {10, 20, 30, 40, 50};
    int n = 5, i;

    // Traversal
    printf("Original array: ");
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }

    // Insertion at index 2
    int pos = 2, val = 25;
    for(i = n; i > pos; i--) {
        arr[i] = arr[i - 1];
    }
    arr[pos] = val;
    n++;

    printf("\\nAfter insertion: ");
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }

    // Deletion at index 3
    for(i = 3; i < n - 1; i++) {
        arr[i] = arr[i + 1];
    }
    n--;

    printf("\\nAfter deletion: ");
    for(i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }

    // Searching for 30
    int key = 30, found = 0;
    for(i = 0; i < n; i++) {
        if(arr[i] == key) {
            printf("\\nElement %d found at index %d", key, i);
            found = 1;
            break;
        }
    }
    if(!found) printf("\\nElement %d not found", key);

    return 0;
}

Interview Questions

What is array traversal?

Accessing and processing each element of an array using loops.

How is insertion done in arrays?

Elements are shifted right from the desired index, and the new value is placed in the gap.

Why is deletion costly in arrays?

Because all elements after the deleted one need to be shifted left to fill the gap.
