Standard Library String Functions in C

Overview

C provides a set of standard library functions under <string.h> for efficient string manipulation. These functions perform tasks such as copying, concatenation, comparison, searching, and length measurement of strings.

Pro Tip

Always include #include <string.h> at the top of your C file when working with string functions.

Commonly Used String Functions

strlen(str)

Returns the length of the string (excluding the null character).

strcpy(dest, src)

Copies src string into dest.

strcat(dest, src)

Appends src string to the end of dest.

strcmp(s1, s2)

Compares two strings and returns 0 if equal, >0 or <0 otherwise.

More String Functions

  • strncpy(dest, src, n) – Copies first n characters of src into dest.
  • strncmp(s1, s2, n) – Compares first n characters of two strings.
  • strchr(str, ch) – Returns a pointer to the first occurrence of ch.
  • strrchr(str, ch) – Finds the last occurrence of a character.
  • strstr(str1, str2) – Finds the first occurrence of str2 in str1.
  • strtok(str, delim) – Tokenizes a string using the given delimiter.

Example Code

string_functions.c
#include <stdio.h>
#include <string.h>

int main() {
    char a[50] = "Hello";
    char b[] = "World";

    printf("Length of a: %lu\\n", strlen(a));
    strcat(a, b);
    printf("Concatenated: %s\\n", a);
    
    strcpy(a, b);
    printf("Copied: %s\\n", a);
    
    int cmp = strcmp("abc", "abd");
    printf("Comparison result: %d\\n", cmp);

    return 0;
}

Real-World Applications

Search Engines

Used in text matching, parsing, and indexing.

Validation

Validating email, passwords, usernames using strlen and strcmp.

Database Parsing

Parsing SQL-like queries and fields using strtok.

Interview Questions

What is the difference between strcpy and strncpy?

strcpy copies entire string including '\0', while strncpy copies only up to n characters. Use strncpy for safer buffer handling.

How does strtok work?

strtok splits a string into tokens based on a delimiter and uses internal state to continue across calls. It modifies the original string.
