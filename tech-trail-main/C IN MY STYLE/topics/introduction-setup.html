How to Set Up C Compiler

Overview

To start coding in C, you need a compiler. A compiler converts your C code into machine code that your computer can execute directly. The most commonly used compiler is GCC (GNU Compiler Collection).

Key Point

Installing a compiler is the first step to developing and running C programs on your computer.

Installing C Compiler

For Windows

Download and install MSYS2, then use pacman to install GCC:
pacman -S mingw-w64-x86_64-gcc

For macOS

Open terminal and install Command Line Tools:
xcode-select --install

For Linux (Ubuntu/Debian)

Open terminal and run:
sudo apt update && sudo apt install build-essential

Your First C Compilation

Save your code in a file hello.c, then compile it with GCC:

Compile C 
gcc hello.c -o hello

Command Explanation:

  • gcc - calls the GCC compiler
  • hello.c - your C source code
  • -o hello - output file name

Interview Questions

What is a compiler?

A compiler converts your human-readable code into machine code that the processor can execute directly.

What compiler is commonly used for C?

GCC (GNU Compiler Collection) is the most popular compiler for C.

How do you compile a C file?

Using the command gcc file.c -o output, where file.c is your source code and output is the compiled executable.
