Data Types in C Programming

Overview

In C programming, data types specify the type of data that a variable can hold. Each data type requires different amounts of memory and has a specific range of values. Choosing the correct data type is crucial for efficient program memory management.

Key Point

Data types define the nature of variables: what kind of data they store and how much space they consume in memory.

Types of Data Types in C

1. Primary Data Types

Basic types like int, char, float, double.

2. Derived Data Types

Formed from primary types: arrays, pointers, functions.

3. Enumeration

Used to assign names to integral constants, defined using enum.

4. User-defined Data Types

Created using struct, union, typedef, and enum.

Primary Data Types Table

Type Size (Bytes) Format Specifier Range
int 4 %d -2,147,483,648 to 2,147,483,647
char 1 %c -128 to 127
float 4 %f ~1.2E-38 to 3.4E+38
double 8 %lf ~2.3E-308 to 1.7E+308

Example

A simple example using different data types:

datatypes.c
#include <stdio.h>

int main() {
    int age = 21;
    char grade = 'A';
    float height = 5.9;
    double balance = 12345.67;

    printf("Age: %d\\n", age);
    printf("Grade: %c\\n", grade);
    printf("Height: %.1f\\n", height);
    printf("Balance: %lf\\n", balance);

    return 0;
}

Real-World Applications

Scientific Calculations

Uses float and double for precision and scale.

Database Systems

Stores various types like int, char, string types efficiently.

Embedded Systems

Uses specific types to control memory usage precisely.

Interview Questions

What is the difference between float and double in C?

float is a single-precision 32-bit floating point while double is a double-precision 64-bit floating point, offering more precision and range.

Why is it important to choose the correct data type?

Choosing the correct type ensures efficient memory usage, prevents data loss, and improves performance.
