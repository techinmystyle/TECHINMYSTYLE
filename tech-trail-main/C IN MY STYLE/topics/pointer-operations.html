Pointer Operations in C

Overview

Pointers in C allow direct memory access and manipulation. Once a pointer holds the address of a variable, various operations can be performed on it. These operations are essential for efficient programming and are especially useful in array manipulation, function arguments, and dynamic memory.

Key Concept

Pointer arithmetic is always performed relative to the size of the data type the pointer points to.

Common Pointer Operations

  • Assignment: Assigning an address to a pointer.
  • Dereferencing: Accessing the value at the memory address.
  • Arithmetic: Incrementing or decrementing pointer values.
  • Comparison: Comparing pointers (e.g., in arrays).

Example: Pointer Arithmetic

pointer_arithmetic.c
#include <stdio.h>

int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr;

    for (int i = 0; i < 5; i++) {
        printf("Value: %d, Address: %p\\n", *ptr, ptr);
        ptr++; // Move to next element
    }

    return 0;
}

Explanation:

  • ptr = arr; assigns the base address of array to pointer.
  • ptr++; increments the pointer to point to the next integer (next 4 bytes).
  • Each iteration accesses and prints a value using *ptr.

Interview Questions

Can we add two pointers?

No, adding two pointers is not allowed. Only subtraction or relational comparison is valid between compatible pointers.

What does ptr++ do?

It increases the pointer to point to the next memory location of its data type (e.g., +4 bytes for int).

Can we compare two pointers?

Yes, two pointers can be compared using relational operators if they point to elements within the same array.
