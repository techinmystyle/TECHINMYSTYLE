Assignment Operators in C

Introduction

Assignment operators in C are used to assign values to variables. The most commonly used operator is =, which assigns the value on the right to the variable on the left. C also provides compound assignment operators that combine arithmetic or bitwise operations with assignment.

Did You Know?

The expression x += 5; is equivalent to x = x + 5; and is often used for concise and efficient coding.

Types of Assignment Operators

  • = : Simple assignment
  • += : Add and assign
  • -= : Subtract and assign
  • *= : Multiply and assign
  • /= : Divide and assign
  • %= : Modulus and assign
  • &= : Bitwise AND and assign
  • |= : Bitwise OR and assign
  • ^= : Bitwise XOR and assign
  • <<= : Left shift and assign
  • >>= : Right shift and assign

Example Program

The following C program demonstrates various assignment operators:

assignment_operators.c
#include <stdio.h>

int main() {
    int x = 10;
    x += 5; // x = 15
    printf("x += 5: %d\\n", x);

    x -= 3; // x = 12
    printf("x -= 3: %d\\n", x);

    x *= 2; // x = 24
    printf("x *= 2: %d\\n", x);

    x /= 4; // x = 6
    printf("x /= 4: %d\\n", x);

    x %= 4; // x = 2
    printf("x %%= 4: %d\\n", x);

    return 0;
}

Common Mistakes

  • Confusing = (assignment) with == (equality comparison)
  • Using uninitialized variables with assignment operators
  • Using /= with a divisor that could be zero

Interview Questions

What is the difference between = and == in C?

= is used to assign a value, while == is used to compare two values for equality.

What is the result of x *= y + 1?

According to operator precedence, it is evaluated as x = x * (y + 1).
