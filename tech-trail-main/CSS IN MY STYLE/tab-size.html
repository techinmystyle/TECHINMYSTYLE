  • z-index
  • zoom

The tab-size property in CSS specifies the width of tab characters (\t). It accepts integer values representing the number of spaces (e.g., 4, 8) or length values like 2em or 20px. The default value is typically 8. This property is particularly useful for displaying code snippets, preformatted text in <pre> elements, or any content where tab characters are preserved. It's often used alongside white-space: pre to maintain formatting in code examples.
